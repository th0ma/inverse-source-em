Public API for inverse_source_em.data.

This subpackage contains all dataset-generation pipelines used in the inverse EM project. The pipelines are grouped into five domains:

  1. Surrogate dataset generation

    • Sampling utilities for surrogate training
    • SurrogateDataGenerator for Esurf/Hsurf datasets

  2. Classification dataset generation

    • Sampling utilities for multi-source classification
    • Unified surrogate-based forward model
    • Full dataset builder for the 1–5 source count task

  3. Regression Problem I (single-source localization)

    • Sampling utilities for 1-source geometry
    • Surrogate-based forward model
    • Dataset builder + PyTorch Dataset class

  4. Regression Problem II (two-source localization)

    • Sampling utilities with geometric priors
    • Feature construction using surrogate forward model
    • Dataset builder with MinMax scaling

  5. Regression Problem III (three-source localization)

    • Geometry curriculum levels
    • Sampling with stage-based constraints
    • Unified forward model
    • Stage-based dataset generation and scaling

Only the high-level public entrypoints are exported here. Internal helpers remain inside their respective modules.

24def sample_sources(N, R=1.0, rho_min=0.05, rho_max=0.95):
25    """
26    Sample N source positions uniformly in an annular region.
27
28    The sampling domain is:
29        rho ∈ [rho_min * R, rho_max * R]
30        phi ∈ [0, 2π)
31
32    The radial coordinate is sampled using sqrt() so that the resulting
33    distribution is uniform with respect to area (i.e., uniform density
34    in the disk annulus).
35
36    Parameters
37    ----------
38    N : int
39        Number of source positions to sample.
40    R : float, optional
41        Cylinder radius. Default is 1.0.
42    rho_min : float, optional
43        Minimum normalized radial coordinate (relative to R).
44    rho_max : float, optional
45        Maximum normalized radial coordinate (relative to R).
46
47    Returns
48    -------
49    rho : ndarray of shape (N,)
50        Radial coordinates of sampled sources.
51    phi : ndarray of shape (N,)
52        Angular coordinates of sampled sources in radians.
53
54    Notes
55    -----
56    - The returned rho values lie in [rho_min * R, rho_max * R].
57    - The sampling is rotationally symmetric and area-uniform.
58    """
59    r2_min = (rho_min / R)**2
60    r2_max = (rho_max / R)**2
61
62    rho = np.sqrt(np.random.uniform(r2_min, r2_max, size=N)) * R
63    phi = np.random.uniform(0.0, 2*np.pi, size=N)
64    return rho, phi

28    """
29    Compute multi-source Esurf and Hsurf using SurrogateWrapper.
30
31    This function evaluates the surrogate forward model for each source
32    in the list and sums the resulting fields to obtain the total boundary
33    fields. The output is formatted as a 4-channel feature tensor suitable
34    for classification tasks.
35
36    Parameters
37    ----------
38    sources : list of dict
39        List of source dictionaries, each with keys:
40            {"rho", "phi", "I"}
41        where:
42            rho : float
43                Radial coordinate of the source.
44            phi : float
45                Angular coordinate of the source.
46            I : float
47                Source strength.
48    theta : ndarray of shape (num_angles,)
49        Observation angles in radians.
50    sur_wrap : SurrogateWrapper
51        Unified surrogate forward model providing:
52            Esurf(rho, phi, theta)
53            Hsurf(rho, phi, theta)
54
55    Returns
56    -------
57    feat : ndarray of shape (4, num_angles)
58        Feature tensor containing:
59            [Re(E_total), Im(E_total), Re(H_total), Im(H_total)]
60
61    Notes
62    -----
63    - All computations are performed in complex128 for numerical stability.
64    - The final feature tensor is returned as float32 for compact storage.
65    - This function is used by the classification dataset generator.
66    """
67
68    # Initialize accumulators
69    E_total = np.zeros_like(theta, dtype=np.complex128)
70    H_total = np.zeros_like(theta, dtype=np.complex128)
71
72    # Superposition of fields
73    for src in sources:
74        rho = src["rho"]
75        phi = src["phi"]
76        I   = src["I"]
77
78        # SurrogateWrapper returns complex arrays
79        E = sur_wrap.Esurf(rho, phi, theta) * I
80        H = sur_wrap.Hsurf(rho, phi, theta) * I
81
82        E_total += E
83        H_total += H
84
85    # Build feature tensor (4, num_angles)
86    feat = np.stack([
87        E_total.real,
88        E_total.imag,
89        H_total.real,
90        H_total.imag
91    ], axis=0)
92
