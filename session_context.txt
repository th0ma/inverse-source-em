PROJECT: inverse-source-em
AUTHOR: Thomas D. Papadopoulos, e-mail: thomas.d.papadopoulos@gmail.com

DESCRIPTION
-----------
Το inverse-source-em είναι ένα scientific ML project για:
- surrogate EM forward models,
- inverse classification του πλήθους πηγών (1 έως 5),
- inverse source localization για 1, 2, 3 πηγές.

CANONICAL RULE
--------------
Το surrogate pipeline έχει ολοκληρωθεί επιτυχώς και δεν αλλάζει στο εξής.
Αποτελεί το reference baseline για όλα τα επόμενα στάδια.

SURROGATE PIPELINE (CANONICAL)
------------------------------
Βρίσκεται στο src/inverse_source_em/surrogate/ και περιλαμβάνει:

1. surrogate.py
   - SurrogateEM
   - SurrogateMLP
   - batch_Esurf / batch_Hsurf
   - Esurf / Hsurf unified API

2. surrogate_wrapper.py
   - PhysicsTM-compatible wrapper
   - Esurf, Hsurf, batch_forward

3. sampling_surrogate.py
   - sample_sources(N, R, rho_min, rho_max)
   - sample_angles(M)

4. generator_surrogate.py
   - SurrogateDataGenerator
   - generate(kind, n_sources)
   - save_npz

Αυτά τα αρχεία είναι canonical και δεν τροποποιούνται.

CLASSIFICATION PIPELINE (ACTIVE DEVELOPMENT)
--------------------------------------------
Βρίσκεται στο src/inverse_source_em/data/ και περιλαμβάνει:

- sampling_classification.py
- generator_classification.py
- dataset_classification.py

Το classification pipeline έχει διαφορετική φυσική από το surrogate:
- χρησιμοποιεί intensity I,
- χρησιμοποιεί dicts για πηγές,
- υποστηρίζει πολλαπλές πηγές,
- δεν μοιράζεται sampling με το surrogate.

CURRENT STATE
-------------
Έχουμε ολοκληρώσει:
- την παραγωγή δειγμάτων για το classification pipeline,
- το classification training (ResNet1D baseline),
- την αξιολόγηση σε test set.

Baseline αποτελέσματα:
- Best validation accuracy: ~0.9841
- Test accuracy: ~0.9846
- Confusion matrix και classification report σταθερά και συμβατά με το παλιό framework.

Το classification pipeline θεωρείται πλέον σταθερό και reproducible.
